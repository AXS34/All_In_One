# 常用变量

* 1. [sin($F)](),F即Frame,表示按当前帧数进行正弦运算,生成的是正弦曲线,可做小球跳动动画(无衰减,想想怎么做有衰减的动画?);

* 2. [sin($PT)](),PT是指当前处理点的序号,表示根据当前点的序号进行正弦运算(T是否有"当前"的含义,即time?);

* 3. [$T](),T即time,当前时间,注意H中24帧=1S,所以在N帧的时候,对应的是N*(1/24);
